# Basic-Restful-API
# Purpose 
A Restful API to fetch data based on the parameters of customer, lat, and long from a google places API.


# Installation
```
$ git clone https://github.com/nolansingroy/basic-Restful-API
$ npm install 
$ npm run 
```
* Vist localhost:3000/
